# FrontEnd

## 프로젝트 소개

이 프로젝트는 **TypeScript**, **React**, **Vite**, **Styled-Components**, **Redux**, 그리고 **Axios**를 사용하여 개발됩니다. 이 조합을 통해 빠르고 효율적인 개발 환경과 현대적인 프론트엔드 애플리케이션 구조를 구현할 예정입니다.

---

## 기술 스택

### 1. **React (TypeScript)**

- 컴포넌트 기반 UI 라이브러리.
- 타입 안정성을 제공하여 코드의 유지보수성과 가독성을 향상.

### 2. **Vite**

- 빠른 개발 서버와 빌드 속도를 제공.
- 최신 ES 모듈을 활용한 경량화된 빌드 도구.

### 3. **Styled-Components**

- CSS-in-JS 방식으로 컴포넌트 단위로 스타일링.
- 테마 및 전역 스타일 관리를 지원.

### 4. **Redux**

- 애플리케이션 상태를 중앙 집중식으로 관리.
- Redux Toolkit을 활용하여 효율적인 상태 관리 구현.

### 5. **Axios**

- HTTP 요청 및 응답을 처리하는 라이브러리.
- REST API와의 통신을 간단하게 구현.

---

## 프로젝트 목표

1. **최신 프론트엔드 기술 스택 활용**: Vite와 TypeScript를 기반으로 효율적이고 안정적인 개발 환경을 구축.
2. **상태 관리의 일관성**: Redux를 통해 애플리케이션 상태를 중앙에서 관리하여 예측 가능한 동작 구현.
3. **컴포넌트 기반 스타일링**: Styled-Components를 활용해 재사용 가능한 UI 컴포넌트 개발.
4. **API 통신 최적화**: Axios를 활용해 RESTful API와의 원활한 데이터 통신 구현.
